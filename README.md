# e-commerce-website

## Overview

This repository contains the backend code for an e-commerce project. Some Api's have been implemented using NodeJS and ExpressJS with MYSQL serving as the database for the project with the help of Sequelize ORM. Authentication part has been implemented using JWT.

## Setting up your project

- Install Node in your system (https://nodejs.org/en/download/).
- Run `npm install` to install all the dependencies.
- Create MySQL Apache server (https://www.apachefriends.org/download.html).
- Create a database using `phpMyAdmin` dashboard.
- Open `localhost:3000/sync` for creating tables in your database.
- Download Postman on your system for hitting API's (https://www.getpostman.com/collections/839448357cca75bc424b).
